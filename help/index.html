How to use

Navigation

Use the navigation bar at the top to switch between the form and preview, change the template style, and change the theme and colors.

Export

Click the Export button to download a zip file that contains all the HTML, CSS, and JavaScript necessary to host the website.

Import

Load your progress from a previous session by uploading the user_data.json file included in the exported zip folder.
